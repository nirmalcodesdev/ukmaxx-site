# UKMAXX Master Label System

Deliverables included:
- `UKMAXX-master-label-system.svg` (master editable source)

## Editable fields
- `{{PRODUCT_NAME}}`
- `{{STRENGTH}}`
- `{{VOLUME_ML}}`
- `{{BATCH_CODE}}`
- `{{PURITY_PERCENT}}`
- `{{EXPIRY_DATE}}`
- `{{VERIFICATION_TYPE}}`
- `{{QR_CODE}}`
- `{{COA_URL}}`

## Non-editable system elements
- UKMAXX logo mark
- UKMAXX brand name
- Layout structure
- Typography system
- Colour palette

## Sizes mapped in master
- Front label: 50mm × 30mm
- Verification label: 50mm × 20mm
- Tamper seal: 25mm × 25mm

The SVG uses a 12px-per-mm working scale:
- 50mm = 600px
- 30mm = 360px
- 20mm = 240px
- 25mm = 300px

## Figma
Figma accepts SVG import directly. Import `UKMAXX-master-label-system.svg` and duplicate each group as needed.

## PNG export template
From Figma (or Illustrator/Inkscape), export each group at 300 DPI with transparent background as needed:
- `front_label_50x30`
- `verification_label_50x20`
- `tamper_seal_25x25`
